# Student Grade Manager

A console-based Java application to manage student grades using Stream API.

## Features
- Add students with name, subject and marks
- Display all students
- Sort students by marks (highest first)
- Filter students scoring above 60
- Calculate average marks

## How to Run
1. Clone the repo
2. Open in IntelliJ or any Java IDE
3. Run `ecer.java`

## What I Learned
- Java Stream API — filter, sorted, mapToInt, average
- Comparator for custom sorting
- OOP principles — encapsulation, single responsibility

## Tech Stack
- Java 17+
